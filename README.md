# littlemessage
# 可以重复发短信

### 如果填写发送号码，会发到这个指定的手机上。如果不填写发送号码，每一次发送会随机产生一个有效的手机号码。

### 发送的次数，如果太大，只会发送少量次。如果你知道自己在干嘛，可以给发送次数前加root,比如"root38"
